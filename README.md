# خواننده پیام فارسی 🎙️

## روش ۱: ساخت APK با GitHub (رایگان، ۵ دقیقه)

### مرحله ۱ - آپلود کد به GitHub:
1. به [github.com](https://github.com) بروید و لاگین کنید
2. دکمه **New repository** را بزنید
3. نام `farsi-message-reader` بگذارید و **Create** کنید
4. فایل ZIP را extract کنید
5. همه فایل‌ها را drag & drop روی صفحه GitHub کنید
6. دکمه **Commit changes** را بزنید

### مرحله ۲ - دانلود APK:
1. روی تب **Actions** در GitHub کلیک کنید
2. آخرین workflow را باز کنید (چند دقیقه طول می‌کشد)
3. زیر **Artifacts** روی **FarsiMessageReader-APK** کلیک کنید
4. فایل ZIP دانلود می‌شود، آن را extract کنید
5. فایل `app-debug.apk` را روی گوشی کپی و نصب کنید

## روش ۲: نصب روی گوشی
1. فایل APK را به گوشی منتقل کنید
2. در Settings > Security گزینه **Unknown sources** را فعال کنید
3. APK را باز کرده و نصب کنید

## پس از نصب:
1. دسترسی‌ها را فعال کنید (SMS، میکروفون، اعلان‌ها)
2. دکمه **تست صدا** را بزنید
3. آماده است!
